# Employee Task Management System

A full-stack application built using Spring Boot and React.

## Tech Stack
### Backend
- Java
- Spring Boot
- MySQL
- JPA / Hibernate

### Frontend
- React
- Axios
- Bootstrap
- Chart.js

## How to Run
### Backend
- Run Spring Boot application
- Server runs on http://localhost:8080

### Frontend
- npm install
- npm start
- App runs on http://localhost:3000
